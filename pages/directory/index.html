目录结构

完整版 (opens new window)目录结构

├── sql  # 数据库文件
│   ├── datasources.sql  # 数据源表SQL文件
│   ├── template.sql     # 模板表SQL文件
├── static  # 静态资源
├── src
│   ├── java  # 源码
│   ├── resources  # 自定义通用组件
│   │   ├── codegen             # 静态资源
│   │   ├── logback             # 系统日志格式配置
│   │   ├── META-INF            # 元信息
│   │   ├── templates           # 生成模板
│   │   ├── hikari.properties   # 数据库属性
│   │   ├── logback.xml         # logback 配置
│   │   ├── mybatis-config.xml  # mybatis 配置文件
├── .gitignore          # git 提交忽略文件
├── CHANGELOG.md        # 版本更新日志
├── CODE_OF_CONDUCT.md  # 行为准则
├── CONTRIBUTING.md     # 贡献指南
├── LICENSE  # 证书
├── README.md  # README
└── pom.xml  # pom 配置
